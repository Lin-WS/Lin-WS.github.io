Q&A

  • New 机电耦合系数越高换能器发射带宽越宽？

Answer

  • New 高频声波穿透能力差？

穿透能力主要看波长，对于高频来说，波长比较短，可能与物体的尺寸相当甚至更小，因此穿透能力就弱；低频波长长，比物体的尺寸大的话能很容易地穿透。光也是一样的，红光波长长，频率低，因此穿透能力强。

  • New 加速度计横向干扰？

横向灵敏度是指加速度计在承受一个正交于敏感方向的加速度信号时，其输出与输入加速度值之比。横向灵敏度与轴向灵敏度之比称为”横向灵敏度比”，它是衡量传感器性能的重要指标。

横向干扰指的是加入一个正交于加速度计敏感方向的激励。

  • New 尝试在钹式加速度计底面加入横向干扰，即加入一个 r 方向的加速度激励，大小为 $1m/s^{-2}$ ，但其100Hz响应非常高(不正常)？

Answer

  • Done $\epsilon_0$ 、$\epsilon_r$、$\epsilon$？

$\epsilon_0$ 为真空介电常数，其值为 $8.854×10^{-12}V/m$，$\epsilon_r$ 称为相对介电常数，即某介质相对真空介电常数的值，单位为1，比如水$\epsilon_r=76.58$，意思就是水的介电常数是真空的76.58倍，$\epsilon$ 就叫做介电常数，是指某一个电介质的实际的介电常数值，单位自然是$V/m$ ，三者的关系为 $\epsilon = \epsilon_0\epsilon_r$ 。

  • Done 特征频率的作用？

可以了解共振频率，还可以知道结构哪些地方是波峰和波谷。

  • Done 二维极坐标指向性图如何显示三维指向性？

一般来说，二维极坐标指向性图只能指示方向，不能确定物体的深度，而且指向的平面就是换能器所处的平面，三维指向性图可以指式物体的具体位置，包括方向和z轴的深度。

  • Done 一阶固有频率设置为 9000Hz？

并无此类说法，固有频率是物体本身的参数，只与结构、材料等有关，在 Comsol 中无法将其作为一个参数随意指定，只能通过修改模型的结构、材料等改变模型的固有频率。

  • Done 换能器的一般仿真流程？

在设计一个换能器时，首先要明确所设计的换能器的用途，比如用作发射或者接收，然后要确定换能器的特性，或者说优点，比如宽带、低频、矢量等等，在仿真时，需要时刻围绕着这些指标操作，比如仿一个用作发射的宽带换能器，除了它的特征频率、电声特性等，应着重对它的发送电压响应、频带宽度进行分析。即在仿真时，除了换能器的基本参数以外，应根据自己所要求的指标，有目的的仿真。

  • Done 测试通道的一致性？

一致性包括灵敏度的一致性和相位的一致性，灵敏度的一致性表示在用 X 通道测量时得到一个电压值，将换能器旋转 90° 使用 Y 通道测量时得到另一个电压值，一致性需要保证这两个电压值尽可能接近，理想情况是它俩相等；而相位的一致性指的是 X 和 Y 通道同时测量时得到电压值的时间差要尽可能小。在测试水听器时，需要对一致性进行测量。

  • Done 在自由圆柱体特征频率一例中，243.5Hz 为什么是二阶频率？它的一阶频率是啥？

正常情况下，对仿真出来的特征频率分阶数是从第一个“正常”的特征频率开始的，比如问题中说的自由圆柱体仿真结果是第一个特征频率为纯虚数，属于“不正常”，因此不能算作一阶，而第二个频率 243.5 看起来“正常”，再结合它的模态，可以判断它是一阶特征频率，有时候还会出现特征频率是实数，但却非常小，数量级已经到 $10^{-3}$ 甚至 $10^{-4}$ 以上了，这些也都不是“正常”的频率。所以即使Comsol 案例文档说 243.5 是二阶特征频率，我们还是认为它是一阶特征频率。

  • Done 在 Comsol 仿真特征频率时为什么总是伴随一些复数特征频率？如下图。

它们是 Comsol 计算求解的过程中得到的，并非我们仿真需要的特征频率，一般都直接忽略。

  • Done 在 Comsol 中定义变量表达式时，acpr.rho 等等变量在哪能查到？

需要在模型开发器中单击显示更多选项(一个小眼睛的图标)，勾选方程视图复选框，之后便可以在各个物理场节点下看到方程视图栏，其中就包含了软件所定义的变量及表达式。

  • Done  宽频和高灵敏度是一对矛盾？

对于水听器来说，高灵敏度也称之为高输出响应，在能量一定的情况下，频带越宽，其谐振频率越高，而在响应平稳的部分下工作时，同一频率对应的响应与谐振频率呈负相关，即能量一定时，谐振频率越高，响应越低。

  • Done 发射换能器的谐振频率和工作频率的关系？

二者没有直接关系。谐振频率指的是换能器的固有频率，由材料参数决定，而工作频率是人为设置的。一般情况下，发射换能器是工作在谐振频率附近的，这样可以获得更大的电压响应，但是也不排除工作在远离谐振频率的情况。
